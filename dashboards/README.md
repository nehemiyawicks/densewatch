# dashboards (M3)

Opinionated Grafana dashboards shipped as JSON:

1. **Rack heatmap** - power density + inlet/coolant temp per rack
2. **Capacity planner** - stranded vs. used power; "how many more GPUs fit"
3. **Job impact** - for a running job: racks, power drawn, thermal envelope, throttle risk
4. **Cooling health** - CDU flow/ΔT/pressure vs. heat load; early-warning on shortfall

Not built yet.
